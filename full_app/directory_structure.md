# Directory Structure


Ordinary, default structure of a PhoneGap app:

    /app_name
    ./hooks
    ./merges
    ./platforms
    ./plugins
    ./www


And in our `www` directory we have our app-specific structure:

    css/
        app.css
        ionicons.css
    fonts/
        ionicons....
    img
    locales
    res/
        icon/
        screen/
    splash
    src/
        famous/
        lib/
        lib2/
        models/
        views/
        config.xml
        credentials.json
        device_ready.js
        history.js
        main.js
        router.js
        text.js
        utils.js
    .gitignore
    config.xml
    icon.png
    index.html
    splash.png
